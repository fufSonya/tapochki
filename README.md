Проект реализует инструмент командной строки для учебного конфигурационного языка. Он парсит конфигурации с числами, массивами и словарями, поддерживает объявление и вычисление констант, и конвертирует их в формат TOML.

## Сделано
- Реализован токенайзер (Lexer) для чисел, массивов и словарей, а также стрелок.
- Реализован парсер (Parser) для всех конструкций языка и вложенных структур.
- Добавлен CLI (cfgtotoml.py), который читает конфигурацию из stdin и сохраняет результат в TOML.
- Написаны тесты для токенайзера и парсера с использованием pytest.
- Примеры конфигураций из разных областей подготовлены для проверки работы конвертера.

## Запуск CLI
python cfgtotoml.py --out output.toml
